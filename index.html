Most common Git commands

  • $ git status - list all new or changed files that need to be fixed
  • $ git config --global user.name "[name]" - set the name that will be shown in author input
  • $ git init [project name] - create a new local named repository
  • $ git clone [URL] - download a repository with whole its history of changes
  • $ git branch [branch name] - create a new branch
  • $ git pull - load the history from remote repository and unite it with the local one
  • $ git push - load all changes of local branch to remote repository
